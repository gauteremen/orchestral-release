# Orchestral — Mac-nedlastinger (offentlig)

Kun `latest.json` skal brukes fra main. **DMG lastes ned fra [Releases](https://github.com/gauteremen/orchestral-release/releases).**

## Viktig

Hvis du klikker DMG-filen i fil-listen på main og laster ned via «Raw», får du en liten LFS-pekerfil (~130 byte). macOS sier da **«Diskfilen er ødelagt»**.

**Riktig:** [siste release](https://github.com/gauteremen/orchestral-release/releases/latest) eller lenken i [latest.json](./latest.json).

Appen sjekker [latest.json](./latest.json) for nye versjoner.
